**Title: Multi-Scale Residual Scattering-Flow (MSRSF) with Contrastive Nuisance-Invariant Embeddings**

The current VCSF pipeline relies on a regressor to condition the flow, which is susceptible to propagation errors if the regressor is uncertain. I hypothesize that we can improve OoD detection by shifting from a purely generative conditional density estimation to a **hybrid generative-contrastive approach**. 

1. **Contrastive Feature Learning:** Instead of relying solely on the regressor for nuisance marginalization, we will train a Siamese-style encoder (using the WST coefficients as input) with a **Triplet Loss** objective. We will define "positive" pairs as maps with the same cosmology but different nuisance realizations, and "negative" pairs as maps with different cosmologies. This forces the latent space to explicitly cluster maps by their physical state while discarding nuisance-driven variance, creating a more robust conditioning manifold for the Normalizing Flow.
2. **Multi-Scale Residual Analysis:** Given that hydro-code differences manifest at specific small scales (1-10 arcmin), we will augment the WST feature vector with **residual-based Minkowski Functionals (MFs)**. By computing MFs on the difference between the observed map and a "Gaussianized" version of the same map (generated via phase-randomization), we isolate the non-Gaussian structural signal from the Gaussian power spectrum.
3. **Flow-based Anomaly Scoring:** We will replace the standard NLL with a **Likelihood-Ratio Test (LRT)** score: $S(x) = \log p(x | \text{InD}) - \log p(x | \text{Background Noise})$. By training a secondary, lightweight flow on noise-only maps (simulating the instrument noise floor), we can penalize scores that are driven by noise fluctuations rather than structural anomalies. 

This approach addresses the "memorization" observed in Iteration 3 by forcing the model to learn a contrastive representation of the physical manifold, rather than just mapping parameters to likelihoods, and provides a more rigorous statistical framework for distinguishing hydro-code signatures from noise.
